# Developed a responsive e-commerce web application frontend using React with component-based architecture.

Implemented client-side routing using React Router for seamless navigation across pages like Home, Product Details, Cart, Wishlist, Checkout, and Account.

Built reusable UI components and structured layout for scalable application design.

Integrated core e-commerce features including product listing, product detail view, shopping cart, and wishlist functionality.

Designed authentication pages (Login & Signup) and user account interface.

Created error handling with a custom Not Found page for better user experience.

Currently working on backend integration for authentication, database connectivity, and order processing.
